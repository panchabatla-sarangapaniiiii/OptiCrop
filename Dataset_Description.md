## Dataset Overview

The dataset contains soil nutrient and environmental parameters used to recommend the most suitable crop for cultivation. It is widely used for crop recommendation, agricultural analytics, and machine learning classification tasks.

The target variable represents the crop that is best suited for the given environmental and soil conditions.

---

## Features

| Feature     | Description                 | Range           |
| ----------- | --------------------------- | --------------- |
| N           | Nitrogen content in soil    | 0 – 140         |
