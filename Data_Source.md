## Dataset Description

The dataset contains soil nutrient measurements and environmental parameters used to determine the most suitable crop for cultivation.

Each record consists of:

* Nitrogen (N)
* Phosphorous (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

The target variable is:

* **Label** – Recommended crop name
